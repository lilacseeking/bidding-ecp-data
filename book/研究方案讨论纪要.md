# 电力物资需求预测研究方案讨论纪要

> 日期：2026-07-04 ~ 2026-07-05
> 数据来源：国网冀北电力 2020-05 ~ 2026-06 招标采购数据
> 目标：基于 VMD-LSTM-CatBoost 的电力物资月度需求量预测，撰写期刊论文

### 1.3 核心矛盾

- 物资种类极多（4,540 种），但绝大多数仅出现 3-10 次
- VMD 分解需要足够长的连续时间序列才有意义
- 直接用 VMD-CNN-LSTM 预测单种物资不可行

## 二、六条可行研究路径

| 路径 | 名称 | 创新性 | 可行性 | 说明 |
|:----:|------|:------:|:------:|------|
| A | 大类聚合 + VMD-CNN-LSTM | 中 | 高 | 将 4,540 种映射到 12 大类，大类时序 51 点充分 |
| B | 层次化预测 (Hierarchical) | 中高 | 中 | 顶层大类→底层物资，两层解耦 |
| C | 物资聚类 + 面板数据建模 | 高 | 中 | 从物资名称提取结构化特征，聚类后做面板预测 |
| D | 两阶段概率预测模型 | 高 | 中 | 发生概率 + 条件数量，适配稀疏数据 |
| E | 少样本学习 / 元学习 | 很高 | 低 | MAML 跨物资迁移，实现难度大 |
| F | 外部因素驱动的混合模型 | 中 | 中 | 投资计划 + 季节性 + LightGBM |

**最终选择：路径 A（大类聚合 + VMD）与路径 D（个体级两阶段）的混合方案**——VMD 在大类级捕获宏观趋势，CatBoost 在个体物资级融合多特征做预测。

## 三、VMD-LSTM-CatBoost 三层架构

### 3.1 架构总览

```
第一层 — VMD 分解
  对大类的月度总需求时序做 VMD 分解
  → IMF₁ (趋势) + IMF₂ (周期) + IMF₃ (噪声) + 残差

第二层 — LSTM 时序预测
  对每个 IMF 分量独立训练 LSTM
  → 预测各分量的未来值

第三层 — CatBoost 多特征融合
  输入: VMD 分量预测值 + 物资属性特征 + 时间特征 + 历史统计
  输出: 个体物资 × 月份的 demand_quantity
```

### 3.2 跨物资信息共享机制

核心原理：**CatBoost 不是按物资单独建模，而是把所有物资的所有月份混在一起训练一个模型**。

物资即使只有 3 个月的历史，其完整的属性画像（电压等级、截面、型号、材质等）可以让它"搭便车"——同类物资的规律就是它的规律。

```
需求量 = 批次规模效应（98%方差）+ 物资个体效应（2%方差）
         ↑ VMD+LSTM捕获              ↑ 物资属性+CatBoost分配
```

### 3.3 特征体系

| 特征层 | 包含因子 | 数据来源 |
|--------|---------|---------|
| 时间序列特征 | lag1/2/3/12、滚动均值、历史峰值、需求间隔 | 已有 |
| 物资属性特征 | 大类、电压等级、截面、型号、芯数、铠装、阻燃 | bid_items 解析 |
| 招标批次特征 | 年份、月份(sin/cos)、批次序号、公告类型、城市 | 已有 |
| 外部宏观因子 | 新增变电容量、铜/铝期货价格 | 需采集 |

## 四、影响因子评估

### 4.1 用户提出的 7 个因子

| 因子 | 判断 | 理由 |
|------|:----:|------|
| 历史需求量 | ✓ 保留 | 时序预测的基础，已有 |
| 工程投资量 | ✓ 保留 | 电力物资是计划驱动的，投资额决定总盘子 |
| 设备进价成本 | △ 改为原材料价格 | 中标价滞后于决策，原材料期货价格是更灵敏的先行指标 |
| 负荷增长量 | △ 改为新增变电容量 | 负荷→物资的因果链太长，新增变电容量是更直接的工程驱动力 |
| 台风每月次数 | ✗ 删除 | 冀北不处台风路径区 |
| 雷击每月次数 | ✗ 删除 | 驱动的是抢修物资，不在招标采购数据中体现 |
| 暴雨每月次数 | ✗ 删除 | 驱动的是应急物资，不在招标采购数据中体现 |

### 4.2 最终推荐因子体系

**内部因子（可直接提取）：**

- 历史需求量 lag1/2/3/12
- 滚动统计：3 月均值、6 月均值、历史峰值
- 距上次需求间隔、需求集中度
- 物资大类、结构化参数（电压等级/截面/型号/材质）
- 该物资在大类中的份额、同大类物资月度总量
- 年份、月份 sin/cos 编码、招标批次序号
- 公告类型（招标 vs 采购）、所在城市

**外部因子（需采集）：**

- 冀北新增变电容量（季度，国家能源局）
- 铜/铝期货价格月均价（日频→月均，上期所 SHFE）

### 4.3 为何删除"电网基建投资完成额"和"工业增加值"

三者都反映行业景气，但位于因果链的不同位置：

```
工业增加值 → 用电需求 → 电网投资 → 新增变电容量 → 物资采购
  (最远端)                                         (最近端)
```

保留"新增变电容量"原因：
1. 与预测目标的因果距离最短——它是物资需求的直接工程驱动力
2. 与铜铝价格完全正交（物理量 vs 价格量），无共线性问题
3. 论文叙事更聚焦，三个维度各司其职

## 五、TOP20 物资分析及 6 种预测目标选择

### 5.1 评估框架

| 维度 | 含义 | 为什么重要 |
|------|------|-----------|
| 数据点数 | 非零月份数量 | VMD 至少需要 8-10 点 |
| CV（变异系数） | std/mean | 越低越平稳，LSTM 更容易学习 |
| 最大单月占比 | 单峰/总量 | <30% 表示需求分布均匀 |
| 近年活跃度 | 2024 年以来数据月数 | 近期数据对预测更重要 |
| 品类多样性 | 所属物资大类 | 覆盖不同大类证明泛化能力 |

### 5.2 六种推荐物资

| 排名 | 物资 | 数据点 | CV | 最大月占比 | 品类 | 选择理由 |
|:--:|------|:--:|:--:|:--:|------|------|
| 1 | 控制电缆 | 15 | **0.47** | 12% | 电缆类 | 最平稳，春秋双周期规律 |
| 2 | 电缆保护管 | **17** | 1.09 | 20% | 管材类 | 数据最多，清晰上升趋势 |
| 3 | 通信单元 | 13 | 0.86 | 21% | 通信设备 | 总量最大(1075万)，Q3-Q4周期 |
| 4 | 布电线 | 13 | 1.16 | 30% | 导线类 | 结构性跳变，考验模型鲁棒性 |
| 5 | 电缆接线端子 | 13 | 0.82 | 21% | 金具类 | 稳定上升，配件类代表 |
| 6 | 蝶式绝缘子 | 12 | 0.86 | 21% | 绝缘子 | 中等规模，品类多样性 |

六种覆盖电缆、管材、通信设备、导线、金具、绝缘子六个大类。

### 5.3 各物资月度需求明细

**控制电缆**（CV=0.47，最佳候选）：
需求量 1 万 ~ 8.7 万，分布均匀。每年 3-5 月和 9 月定期出现需求，呈现春秋两季招标规律。VMD 将分解出清晰的年度周期分量。

**电缆保护管**（17 个数据点，最多）：
从 2021 年 2~15 万增长到 2025 年 35~39 万，VMD 趋势分量将完美捕捉这一增长曲线。

**通信单元**（总量 1,075 万，最大）：
密集出现在 Q3~Q4（8-10 月），每年下半年一波，约 12 个月周期性分量。

**布电线**（结构性跳变）：
2024 年 9 月从 10~46 万跃至 257 万，2025 年维持 96~184 万高位，CatBoost 处理突变 + 特征交互的优势场景。

**电缆接线端子**（CV=0.82，稳定型）：
3.9 万稳步增至 19.7 万，单调上升 + 波动小。

**蝶式绝缘子**（中等规模代表）：
1,144 ~ 102,366，2025 年后明显增长，证明模型不只在头部物资有效。

## 七、影响因子体系设计

### 7.1 初始 7 因子评估

用户提出的 7 个候选因子及其评估：

| 因子 | 判断 | 理由 |
|------|:----:|------|
| 历史需求量 | ✓ 保留 | 时序预测的基础，已有数据 |
| 工程投资量 | ✓ 保留 | 电力物资是计划驱动的，投资额决定总盘子 |
| 设备进价成本 | △ 改为原材料价格 | 中标价滞后于决策；原材料期货价格是更灵敏的先行指标 |
| 负荷增长量 | △ 改为新增变电容量 | 负荷→物资的因果链太长（12-18月）；新增变电容量是更直接的工程驱动力 |
| 台风每月次数 | ✗ 删除 | 冀北（唐山/承德/张家口/秦皇岛/廊坊）不处台风路径区 |
| 雷击每月次数 | ✗ 删除 | 驱动的是抢修物资，不在招标采购数据中体现 |
| 暴雨每月次数 | ✗ 删除 | 驱动的是应急物资，不在招标采购数据中体现 |

### 7.2 三轮迭代优化

**第一轮：六因子方案（已废弃）**

包含：历史需求惯性（HDIF）、物资技术规格（MTSF）、采购批次节律（PBRF）、电网投资拉动（GIPF）、原材料成本波动（RMCFF）、区域需求特征（RDPF）

**第二轮：三因子被否决**

| 因子 | 否决理由 | 结论 |
|------|---------|:--:|
| 物资技术规格（MTSF） | CatBoost 验证中物资属性特征重要性接近零，CV=0.47 和 CV=2.24 的物资无法被规格特征区分。"相似规格→相似需求"假设不成立 | 删除 |
| 采购批次节律（PBRF） | 数据泄露（data leakage）问题。预测 2026 年 9 月需求时，"当年第几批招标"是不可知的——批次序号是采购行为发生后才能确定的标签，不能用未来信息预测未来 | 删除 |
| 区域需求特征（RDPF） | 论文聚焦冀北单一地区，无跨区域对比需求。城市级别拆分对冀北公司内部管理有意义，但对"需求量预测"课题不构成独立因子 | 删除 |

**第三轮：五因子方案（当前版本）**

核心原则：只保留**独立于历史需求数据之外**的变量。DSF（需求稀疏度因子）和 SCF（季节周期因子）虽然有效，但本质上是历史数据的派生变换（特征工程），而非真正的外生影响因子，因此不列入影响因子清单。

| 编号 | 因子 | 数据来源 | 获取难度 | 因果维度 |
|:--:|------|---------|:--:|:--:|
| STF | 季节时间因子 | 纯历法计算 | 无 | 制度节律 |
| RMPF | 原材料价格因子 | 上期所 SHFE（铜/铝期货） | 低 | 市场成本端 |
| IEDF | 工业用电需求因子 | 国家能源局月报（全社会用电量） | 低 | 需求侧信号 |
| GIDF | 电网投资驱动因子 | 国家能源局/中电联月报 | 中 | 供给侧工程拉动力 |
| NEIF | 新能源并网容量因子 | 国家能源局季报（风电/光伏装机） | 中 | 新能源政策驱动力 |

### 7.3 特征工程 vs 影响因子的区分

```
特征工程：从历史数据中派生的数学变换
  ├─ lag₁, lag₂, lag₃, lag₁₂ → 自回归信号
  ├─ 滚动均值、历史峰值 → 统计量
  ├─ month_sin, month_cos → 周期编码
  └─ 需求间隔月数、稀疏密度 → 时序结构特征

影响因子：独立于历史需求数据之外的外部变量
  ├─ STF → 日历（不属于"数据"）
  ├─ RMPF → 全球商品市场（不由冀北电网决定）
  ├─ IEDF → 区域经济（不由物资采购决定）
  ├─ GIDF → 投资决策（外生冲击）
  └─ NEIF → 新能源政策（外生冲击）
```

### 7.4 五因子因果链

```
工业用电增速 (IEDF)  ──→ 电网投资决策 ──→ 新增变电容量 (GIDF) ──→ 物资采购量 ←── 原材料价格 (RMPF)
                              ↑                                              ↑
新能源并网 (NEIF)  ──────────┘                                        季节时间 (STF)
                                                                      (制度节律)
                        工程驱动力                                       市场与制度因素
```

五条因果链互相独立：IEDF 和 NEIF 驱动 GIDF，但三者测量的是因果链上不同节点的不同物理量（用电量 kWh vs 装机量 MW vs 投资额 亿元）；RMPF 完全外生于电网投资——铜铝价格由全球商品市场决定；STF 是制度节律，与其他四个经济/工程变量不在一个维度。

## 八、外部数据获取计划

### 8.1 数据获取状态

| 因子 | 数据源 | 工具 | 状态 |
|:--:|------|------|:--:|
| STF | 纯计算 | Python sin/cos | ✅ 可生成 |
| RMPF | AKShare → `futures_zh_daily_sina` | SHFE 铜(CU0)/铝(AL0) | ✅ 已验证，5228条从2005至今 |
| IEDF | AKShare → `macro_china_society_electricity` | 全国月频用电量 | ✅ 已验证，66条(2020-2026) |
| GIDF | 国家能源局月报 | 电网基建投资完成额 | ⚠️ 数据存在但无免费API，需手动整理 |
| NEIF | 国家能源局季报 | 风电光伏新增装机 | ⚠️ 数据存在但无免费API，需手动整理 |

### 8.2 GIDF/NEIF 手动采集指引

**GIDF 电网投资数据：**
- 数据格式：月度累计值"1-X月全国电网工程投资完成XXX亿元"
- 月频来源：中电联每月发布的《全国电力工业统计数据》
- 可获取渠道：国际能源网 (in-en.com) 搜索"电网工程投资完成"追踪月度新闻通稿
- 降级方案：如有年度数据（国网年报），可做线性插值到月度

**NEIF 新能源装机数据：**
- 数据格式：季度数据"第X季度全国风电新增装机XX GW，光伏新增装机XX GW"
- 季频来源：国家能源局季度新闻发布会通稿
- 含河北省层面数据（公开可查）

### 8.3 执行策略（两步走）

**第一步（立即执行）**：STF + RMPF + IEDF 三个确定可用的因子，批量获取 2020-05 ~ 2026-06 数据，写入 SQLite + 导出 Excel
**第二步（后续补充）**：GIDF 和 NEIF 从新闻通稿/年报中手动整理

### 8.4 数据存储结构

```sql
CREATE TABLE external_factors (
    month            TEXT PRIMARY KEY,  -- YYYYMM
    month_sin        REAL,              -- STF: sin编码
    month_cos        REAL,              -- STF: cos编码
    copper_price     REAL,              -- RMPF: 铜期货月均价(元/吨)
    aluminum_price   REAL,              -- RMPF: 铝期货月均价(元/吨)
    elec_consumption REAL,              -- IEDF: 第二产业用电量(亿kWh)
    elec_yoy         REAL,              -- IEDF: 用电量同比增速(%)
    grid_invest      REAL,              -- GIDF: 电网基建投资(亿元) [待补充]
    wind_capacity    REAL,              -- NEIF: 风电新增装机(MW) [待补充]
    solar_capacity   REAL               -- NEIF: 光伏新增装机(MW) [待补充]
);
```
