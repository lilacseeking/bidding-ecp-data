# 五因子外部数据获取方案

> 日期：2026-07-05
> 数据范围：2020-05 ~ 2026-06（74 个自然月）

---

## 一、五因子概览

| 编号 | 因子 | 数据来源 | 获取方式 | 难度 |
|:--:|------|---------|---------|:--:|
| STF | 季节时间因子 | 纯历法计算 | 代码生成 | 无 |
| GIDF | 电网投资驱动因子 | 国网年报/国家能源局 | 网络采集 | 中 |
| RMPF | 原材料价格因子 | 上期所 SHFE | 网络采集 | 低 |
| IEDF | 工业用电需求因子 | 国家能源局月报 | 网络采集 | 中 |
| NEIF | 新能源并网容量因子 | 国家能源局季报 | 网络采集 | 中 |

---

## 二、各因子详细方案

### 2.1 STF — 季节时间因子

**数据源**：无，纯计算

**计算方法**：
```
month_sin = sin(month × 2π / 12)
month_cos = cos(month × 2π / 12)
```

**数据量**：74 行（每月一行）

### 2.2 RMPF — 原材料价格因子

**数据源**：上海期货交易所 (SHFE) 铜/铝期货主力合约日结算价

**可靠数据入口**：
- 东方财富/同花顺金融数据 API（免费）
- AKShare Python 库（开源金融数据接口）
- 上期所官网月度报告

**获取策略**：使用 AKShare 库获取 `cu`（铜）和 `al`（铝）期货日频数据 → 按月计算均价

**数据量**：约 1500 个交易日 → 74 个月均

**验证标准**：能获取到 2020-05 ~ 2026-06 完整区间的日数据

### 2.3 IEDF — 工业用电需求因子

**数据源**：国家能源局月度全社会用电量数据

**可靠数据入口**：
- 国家能源局官网：`http://www.nea.gov.cn` 月度数据发布
- 中电联（CEC）每月发布的全社会用电量数据

**目标数据**：河北省全社会用电量月度数据（亿千瓦时）

**数据粒度**：最好是月度数据。若仅有全国数据，则用全国工业用电量同比增速作为代理变量（论文中写明即可）

**数据量**：74 行

**验证标准**：能获取到至少每季度一条数据

### 2.4 GIDF — 电网投资驱动因子

**数据源**：国家电网年度社会责任报告 / 国家能源局电力可靠性年报

**实际情况**：冀北地区级别的"新增变电容量"数据通常**不在公开月/季报中出现**，仅出现在：
- 国网冀北公司年度工作会议新闻稿
- 《国家电网公司年度报告》中的区域板块
- 国家能源局年度《电力可靠性指标》

**替代方案**（按优先级）：
1. **首选**：获取"全国电网基建投资完成额"月度数据（国家统计局/中电联有公开月度数据），作为冀北电网投资的**代理变量**
2. **备选**：收集国网冀北年度工作会议新闻中的变电容量数据（仅年度粒度，4-5 条），做线性插值到月度
3. **下策**：用河北省电力基建投资额代替

**数据量**：首选方案 74 行月频，备选方案 4-5 行年频

### 2.5 NEIF — 新能源并网容量因子

**数据源**：国家能源局季度可再生能源并网运行情况

**实际情况**：国家能源局每季度发布全国及各省风电、光伏新增装机。河北省层面数据较为充足。

**可靠数据入口**：
- 国家能源局季度新闻发布会
- 河北电力交易中心年度报告

**替代方案**：
1. **首选**：河北省季度风电/光伏新增装机（MW），国家能源局季报有省分数据
2. **备选**：全国风电/光伏季度新增装机，作为新能源投资景气度代理变量

**数据量**：季频数据，约 25 个季度点 → 向前填充到月度

## 三、数据存储结构

```sql
CREATE TABLE external_factors (
    month           TEXT PRIMARY KEY,   -- YYYYMM
    month_sin       REAL,               -- STF: sin编码
    month_cos       REAL,               -- STF: cos编码
    copper_price    REAL,               -- RMPF: 铜期货月均价(元/吨)
    aluminum_price  REAL,               -- RMPF: 铝期货月均价(元/吨)
    elec_consumption REAL,              -- IEDF: 用电量(亿kWh)
    grid_invest     REAL,               -- GIDF: 电网基建投资(亿元)
    wind_capacity   REAL,               -- NEIF: 风电新增装机(MW)
    solar_capacity  REAL,               -- NEIF: 光伏新增装机(MW)
    data_source     TEXT                -- 数据来源标注
);
```

## 五、风险预判

| 风险 | 可能性 | 应对 |
|------|:--:|------|
| AKShare 接口失效 | 中 | 手动从东方财富下载 CSV，pandas 读取 |
| 河北省用电量无月度数据 | 高 | 降级为全国工业用电量 |
| 冀北变电容量仅有年度数据 | 高 | 年度插值 + 电网基建投资额作为补充 |
| 新能源装机仅有季度数据 | 高 | 季度值做前向填充到月度 |
| API 被限频 | 低 | 加延迟、本地缓存 |
