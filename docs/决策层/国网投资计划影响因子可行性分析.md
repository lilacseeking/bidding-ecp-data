# 国网投资计划与项目安排作为影响因子的可行性分析

## 一、国网投资计划数据可获得性

### 1.1 现有公开数据源

| 数据维度 | 数据源 | 可获得性 | 格式 | 频率 |
|---------|--------|---------|------|------|
| 年度投资总额 | 国网年度工作会议/社会责任报告 | ✅ 可获取 | 金额(亿元) | 年度 |
| 总部集中采购批次安排 | ECP2.0 平台公告 | ✅ 可获取 | 批次清单(约104批/年) | 年度预安排 |
| 输变电设备招标批次 | ECP2.0 招标公告 | ✅ 已采集 | 招标公告明细 | 6批次/年 |
| 特高压项目批次 | ECP2.0 | ✅ 已采集 | 设备/材料/服务 | 6批次/年 |
| 智能电表批次 | ECP2.0 | ✅ 已采集 | 约3批次/年 | 每年 |
| 省公司投资计划 | 各省公司官网/媒体通稿 | ⚠️ 部分可获取 | 金额(亿元) | 年度 |
| 月度投资完成额 | 国家能源局/中电联月报 | ⚠️ 需手动整理 | 金额(亿元) | 月度 |
| 电网基建投资月度数据 | 中电联《全国电力工业统计数据》 | ⚠️ 延迟1-2月 | 表格 | 月度 |

### 1.2 关键数据来源详情

#### 来源1：ECP2.0 批次预安排（已可采集）

2026年度约104个批次，包括：
- 输变电设备：6批次 (2月/3月/6月/7月/9月/11月)
- 特高压设备/材料：各6批次
- 电源项目：4批次
- 数字化项目：4批次
- 智能电表：3批次 (3月/6月/10月)

#### 来源2：月度电网投资完成额（需新建采集）

中电联每月发布《全国电力工业统计数据》，包含：
- 电网基建投资完成额（累计值 → 可推导月度值）
- 新增110kV及以上变电设备容量
- 新增110kV及以上输电线路长度

AKShare 已有接口：`ak.macro_china_power_industry()` 或可扩展。

#### 来源3：国网年度工作会议（手动整理）

每年1月召开，发布：
- 上年度投资执行情况
- 本年度投资计划总额
- 重点工程安排
- 各省公司投资分配

## 二、可作为影响因子的具体指标

### 2.1 月度级因子（最优粒度）

| 因子名称 | 英文名 | 数据含义 | 采集方式 |
|---------|--------|---------|---------|
| 当月电网投资完成额 | `monthly_grid_invest` | 全国电网基建投资月度完成额 | 中电联月报+AKShare |
| 当月招标公告数量 | `monthly_bid_notice_cnt` | 国网ECP当月发布的招标公告数 | 已采集(ECP DB) |
| 当月中标金额合计 | `monthly_winning_amount` | 当月中标公告金额总和 | ECP中标公告解析 |
| 当月招标批次密集度 | `bid_batch_density` | 当月是否有招标批次发布(0/1或0-3) | ECP批次预安排 |

### 2.2 年度级因子

| 因子名称 | 英文名 | 数据含义 |
|---------|--------|---------|
| 年度电网投资计划 | `annual_invest_plan` | 国网年度计划投资额 |
| 年度投资同比增长率 | `annual_invest_yoy` | 本年计划/上年计划 - 1 |
| 本年在批次序列中的位置 | `batch_position_ratio` | 当前月份在本年批次计划中的进度(0-1) |

### 2.3 省公司级因子（针对具体省公司数据）

| 因子名称 | 英文名 | 数据含义 |
|---------|--------|---------|
| 省公司年度投资额 | `province_invest_plan` | 具体省份的电网投资计划 |
| 省公司在ECP的招标活跃度 | `province_bid_activity` | 该省公司当月公告数/总公告数 |

### 2.4 衍生/交叉因子

| 因子名称 | 英文名 | 计算方法 |
|---------|--------|---------|
| 投资执行进度偏差 | `invest_gap` | 实际投资 - 计划投资 |
| 批次前移/延后指标 | `batch_shift` | 实际招标月份 vs 预安排月份的偏差 |
| 省公司需求份额 | `province_demand_share` | 该省物资需求/全国物资需求 |

## 三、当前ECP数据中隐含的"投资计划"信号

### 3.1 已有可用信号

当前数据库中已经包含了一些隐含投资计划的数据：

```sql
-- 月度招标公告趋势（可反映采购节奏）
SELECT substr(notice_publish_time,1,7) as ym, COUNT(*)
FROM bid_notices GROUP BY ym ORDER BY ym;

-- 月度物资需求总量趋势
SELECT demand_month, SUM(demand_quantity)
FROM material_demand_total GROUP BY demand_month;

-- 各省采购活跃度
SELECT project_org_name, COUNT(DISTINCT notice_id)
FROM bid_items GROUP BY project_org_name;
```

### 3.2 批次信号提取

从公告标题中可以提取批次信息，已有正则匹配：`第([一二三四五六七八九十\d]+)次`

```python
# 已有代码在 pipeline.py 中
m = re.search(r"第([一二三四五六七八九十\d]+)次", n.title)
if m:
    batch = f"第{m.group(1)}次"
```

## 四、可行性结论

### 推荐方案：三层因子体系

```
Layer 1: 宏观层 (年度频率)
  ├── 国网年度投资计划额 ← 手动采集+年度更新
  ├── 投资计划 YoY 增长率
  └── "十五五"规划年度目标

Layer 2: 中观层 (月度频率, 核心新增)
  ├── 月度电网投资完成额 ← AKShare/中电联采集
  ├── 当月招标公告数量 ← 已有(ECP数据库)
  ├── 当月批次密集度 ← ECP批次预安排
  └── 投资完成进度偏差 ← 计算

Layer 3: 微观层 (已有, 月频)
  ├── 项目数量/公告数 ← 已有
  ├── 铜/铝期货价格 ← 已有(AKShare)
  ├── 工业用电量 ← 已有(AKShare)
  └── 农历月份编码 ← 已有(计算)
```

### 预期效果

| 指标 | 改善幅度 | 理由 |
|------|---------|------|
| CatBoost Baseline R² | 0.15~0.49 → **0.40~0.65** | 宏观投资数据提供需求总量的上界/趋势 |
| 月度需求峰值预测 | 显著改善 | 批次密集度直接对应采购波峰 |
| 年度拐点判断 | 中等改善 | 投资计划提供年度总量约束 |

### 实施建议

1. **P0 (立即)**：添加 `monthly_bid_notice_cnt` 和 `bid_batch_density`（数据已在ECP DB中）
2. **P1 (1-2周)**：集成 `monthly_grid_invest`（需开发中电联数据采集）
3. **P2 (1月+)**：补充年度投资计划手动采集、省公司级因子
