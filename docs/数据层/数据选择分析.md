# 冀北 vs 国网总部 数据选择分析

## 1. 问题背景

当前模型（VMD-CatBoost）使用 `material_demand_total` 表中的数据（国网级别聚合），针对三种目标物资进行需求预测：

- 交流避雷器
- 电容式电压互感器
- 交流支柱绝缘子

需要评估：是否应该使用冀北省级数据（更精细但更稀疏），还是坚持国网全国聚合数据（更多样本但泛化性待验证）。

---

## 2. 数据库表结构解析

| 表名 | 数据来源 | 记录数 | 物资数 | 关键字段 |
|------|----------|--------|--------|----------|
| `bid_items` | ECP招标明细（含交付地点、项目单位） | 366,101 | ~3,000规格 | `project_org_name`, `material_name`（细粒度规格级） |
| `material_demand_item` | 从bid_items聚合后标准化 | 16,129 | 1,525 | `material_name`（简化名称，如"交流避雷器"） |
| `material_demand_total` | material_demand_item的总和 | 16,127 | 1,525 | 同上，与item基本一致 |

**关键发现**：`material_demand_total` 中的"交流避雷器"等名称是简化/标准化后的物资名称，实际对应 `bid_items` 中数百个细粒度子规格的聚合（如"交流避雷器,AC110kV,102kV,瓷,266kV,不带间隙"等），并聚合了全国所有省份（含国网总部+79家省公司）。

**因此，当前"国网数据"实际上是全国聚合数据，不是狭义的总部直属数据。**

真正按组织主体拆分的数据在 `bid_items.project_org_name` 字段中，包括：
- `国家电网有限公司`（总部直属——几乎无数据：仅3-4个月有值，96%零值）
- `国网冀北电力有限公司`（冀北省公司）
- `国网山东省电力公司`（山东省公司）
- 等79家省/直属公司

### 4.3 总部直属数据几乎不可用

`国家电网有限公司`（project_org_name 精确匹配）的数据：
- 三种物资均为 **3~4个非零月 / 81月**

- 最长连续零值 38个月

**结论**：当前所谓"国网总部"数据本质上是**全国各省数据的大合并**——这是正确的做法。

## 5. 模型可行性分析

### 5.1 VMD分解在冀北数据上会遇到的具體问题

VMD（变分模态分解）是一种全信号分解方法，对数据的完整性和连续性有较高要求：

1. **有效信号点数不足**：冀北数据仅17~28个非零月，VMD需要构造等间距完整序列——在整个75/81个月的时间轴上，65%~72%的月份填充了零值，实际包含信息的点仅占28%~37%。VMD在零值序列段的分解退化为对常数的分解，没有物理意义。

2. **连续零值段过长（8~11个月）**：VMD通过交替方向乘子法(ADMM)寻找IMF，依赖于信号的局部振荡特性。在连续8~11个月恒为零的区间内，VMD无法提取任何有意义的模态分量（零信号的所有IMFs也趋近于零）。

3. **插值引入伪信息**：如果用线性/样条插值填充空洞，8~11个月的连续空洞插值将产生大量完全捏造的数据，导致VMD分解出虚假的周期性成分。

4. **CV过高（1.0~1.5）**：冀北数据的需求量波动极大（均值88，标准差132），即使在非零月份内也是极度离散的，高频噪声会主导VMD的分解结果。

### 5.2 VMD分解在国网全国聚合数据上的可行性

国网全国聚合数据：
- 非零月比例：61.7%~64.2%，**已接近常规时序数据的下限**
- 最长连续零值：**仅2个月**，这是非常关键的优势
- 52个非零月/81月 = 64%，样本量基本达标

对于VMD分解：
- K=3~5时，需要至少2K+1=7~11个周期的振荡特征，52个有效点勉强足够
- 连续2个月的空洞可以通过简单的线性插值合理填充
- 64%的数据密度在VMD文献中处于可用范围的下沿，但**不是不可用**

**结论**：国网全国聚合数据的VMD分解具有可行性，但处于临界状态，需要注意：
- 模态数K不宜过大（建议K=3，不超过5）
- 插值策略需要审慎（优先用前值填充/简单线性插值，不建议用复杂插值）
- 应同时尝试非VMD的baseline（如直接使用原始序列+CatBoost）

### 5.3 CatBoost baseline 在国网数据上 R^2=0.15~0.49 是否可接受

对于电力物资月度需求预测这类问题：

| 类型 | 参考R^2范围 | 可比性 |
|------|-----------|--------|
| Croston方法（零值率30%+） | < 0.1 | 同类型数据 |
| 电厂备件 XGBoost（零值率80%） | R^2未报告，MAPE=9.25% | 部分可比 |
| 电力物资 KNN（零值率50%） | 0.3~0.5 | 高度可比 |
| 一般供应链月度预测 | 0.3~0.7 | 较高可比 |

结论：
- **R^2=0.15~0.49 处于行业中等偏低但可接受的范围**，并非模型失败的表现
- 电容式电压互感器的R^2=0.49较好（CV=0.61相对较低），说明数据自身规律性强
- 交流避雷器的R^2=0.15~0.20较差可能与其CV=1.12的高离散性和"集中批次招标"特征有关
- **相比冀北数据，R^2会大幅下降**（数据量更少、离散度更高、连续零值更长）

## 6. 文献对比

### 6.1 z文献中的零值率范围

通过文献检索，梳理了类似场景下数据零值率的分布：

| 文献 | 数据场景 | 零值率 | 预测方法 | 效果 |
|------|---------|--------|---------|------|
| 国家基金项目：电力行业有限需求信息下的间断需求预测（姜爱萍） | 电力物资招标 | ~78% | 组合预测 + Hurdle混合零截尾泊松 | 优于Croston |
| State Grid Corp China：FMC+MAPA（2022） | 省级电力公司月度采购 | ~78% | 模糊马尔可夫链 + MAPA | 精度优于ES |
| 电厂备件集成学习预测（MDPI, 2026） | 电厂关键备件 | ~80% | XGBoost + RF | MAPE=9.25% |
| 电力能源供应链 LSTM+小波变换（IAU） | 电力供应链 | ~50% | LSTM + 小波 + RELM | RELM最优 |
| Microsoft Dynamics 365：Croston方法阈值 | 企业供应链 | >80%才触发Croston | Croston/TSB | 适用 |
| 改进k-NN间歇需求预测（EAAI, 2024） | 合成+真实数据 | 变化 | 零值包容k-NN | 优于传统 |

### 6.2 文献启示

1. **零值率35~38%是相对较好的状态**：文献中间歇需求经常以78~80%零值率为标准，我们的国网全国聚合数据（35.8~38.3%）远低于此阈值，可以避免使用Croston/SBA等专门为零星需求设计的方法。

2. **冀北数据的62.7~71.6%零值率处于"中度间歇"区间**：这个级别在文献中通常建议使用Croston/TSB方法而非直接时序分解。直接VMD+CatBoost的风险很高。

3. **国网数据的长处正是VMD所需要的**：
   - 文献中MAPA（多聚合预测算法）的核心思路是通过时间聚合来降低间歇性——国网聚合76个月的数据已经天然实现了这一点
   - 最长连续零值仅2个月，在所有文献中都是优秀水平

4. **CatBoost baseline的R^2在可接受范围**：文献中类似数据的R^2通常在0.1~0.6区间，0.15~0.49不算差。改进的重点应放在特征工程（外部因子引入、滞后特征、滚动统计）而非换数据集。

## 7. 明确的数据选择建议

### 核心结论：坚持使用国网全国聚合数据，不建议切换到冀北

**理由**：

1. **样本量是根本瓶颈**：冀北每类物资仅17~28个非零月，远低于国网的50~52个，且VMD在其中缺乏有意义的信号可分解。

2. **连续零值是VMD的致命伤**：冀北最长连续零值8~11个月（vs国网仅2个月），VMD在全信号分解过程中无法处理如此长的零值段。

3. **地域聚合天然降低离散度**：全国69省的月度需求加总后，CV从1.0~1.5降至0.61~1.12，规律性明显增强。

4. **文献支撑**：35.8%~38.3%的零值率在间歇需求文献中属于"可处理"范围，而62.7%~71.6%已进入需要专门方法（Croston/TSB等）的领域。

5. **工程实用性**：国网全国级别的采购预测模型具有更广泛的适用性——可以为省级公司做拆分，而冀北模型不能推广到其他省份。

### 如果仍然希望探索冀北数据

可以考虑以下方向（但不作为主要方案）：

- **不使用VMD**：直接用CatBoost + 大量外部因子做月度分类（有需求/无需求） + 回归
- **合并相似省份**：将冀北+山东+河北等相邻省份合并为一个"区域"数据集，提高样本量
- **P2P建模**：将省级数据作为"全局模型（国网聚合）+ 省级残差修正"的两阶段模型

### 当前优先任务

在确认使用国网全国聚合数据的前提下：

1. 完善外部影响因子体系（铜/铝价格、工业用电、电网投资、新能源装机等）
2. 特征工程：时序滞后特征（lag 1, 2, 3, 6, 12）、滚动统计（移动平均、移动标准差）
3. VMD分解参数调优（K=3，合适的alpha值）
4. CatBoost超参数搜索（针对R^2=0.15的物资重点优化）
5. 评估去除特定异常月份（如集中批招月份）对模型的影响
