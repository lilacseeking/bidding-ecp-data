# 电力物资需求预测 —— 项目文档导航

基于国网 ECP2.0 平台采购数据的电力物资月度需求量预测研究。
数据来源：国家电网新一代电子商务平台 (ecp.sgcc.com.cn)，2019-11 ~ 2026-07，81 个月。
