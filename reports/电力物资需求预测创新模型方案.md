# 电力物资需求预测 — 未发表创新模型组合可行性分析

## 一、已发表文献的方法学总结

当前（2020-2025）期刊/会议中已出现的电力物资需求预测方法：

```
第一代：统计方法
  ├── ARIMA/SARIMA
  ├──  Holt-Winters 指数平滑
  └── 灰色预测 GM(1,1)

第二代：机器学习
  ├── SVR (支持向量回归)
  ├── Random Forest / XGBoost
  ├── BP 神经网络
  └── SARIMA + BP 混合 ← 2024年已发表

第三代：信号分解 + 深度学习
  ├── EMD + LSTM  ← 2024年会议论文
  ├── VMD + LSTM  ← 《重庆大学学报》2024
  ├── WOA-VMD-LSTM  ← 已发表 (参数优化VMD)
  ├── PSO/SSA-VMD-LSTM  ← 已发表变体
  └── CEEMDAN + BiLSTM  ← 2023-2024年多篇

第四代：新兴架构
  ├── KAN + TCN/BiLSTM/Transformer ← 2025《Energies》SCI
  └── 深度学习综述 ← 2025年综述论文
```

**已发表的"标准范式"**：信号分解(VMD/EMD/CEEMDAN) → 优化参数(WOA/PSO/SSA) → LSTM/BiLSTM预测 → 求和重构。

**未被发表的空白区域较大**，以下是7个方向的分析。

## 二、创新模型组合方案

### 方案1：Croston 分解 + Temporal Fusion Transformer (TFT)
**⭐⭐⭐⭐⭐ 可行性与发表潜力最高**

#### 动机
电力物资月度需求有 **29-31个零值月（占36-38%）**，属于典型的间歇性需求(intermittent demand)。现有的 VMD-LSTM 方法**未专门处理零值膨胀问题**。

Croston 方法将需求分解为两个独立序列：
- **需求发生频率**（demand interval：相邻非零月的间隔）
- **非零需求量**（demand size：非零月的需求数值）

#### 架构
```
         间歇性需求序列 (50/81月非零, 31/81月为零)
                    │
     ┌──────────────┴──────────────┐
     ▼                              ▼
Croston 分解                     TFT 注意力模块
     │                           (Variable Selection
  ┌──┴──┐                         Networks + Gating)
  ▼     ▼                              │
间隔预测  规模预测                      ▼
(分类)   (回归)               ┌─────────────────┐
  │       │                  │ 未来外部因子     │
  │       │                  │ (铜价/铝价/      │
  └───┬───┘                  │  用电量/批次密度) │
      ▼                      └────────┬────────┘
  联合输出:                                 │
  - 预测月份是否有需求 (概率)      ┌────────┘
  - 如果有需求, 预测需求量          ▼
                           TFT Encoder + Decoder
                                    │
                              ┌─────┴─────┐
                              ▼           ▼
                        分位数预测   注意力权重
                        (P10/P50/P90)(可解释)
```

#### 新颖性
- Croston方法此前只用于零售库存预测，**从未应用于电力物资领域**
- TFT的变量选择网络(Variable Selection Networks)可以识别哪些外部因子对需求真正有用
- 可解释的注意力机制可以揭示"哪些历史月份最影响当前预测"
- 分位数预测提供采购决策的不确定性区间

#### 发表可行性
- **高**：间歇性需求预测 + 可解释深度学习 = 强创新点
- 目标期刊：*Applied Energy* / *Energy* / *Electric Power Systems Research*

### 方案2：层次化多物资联合预测 (Hierarchical Forecasting)

**⭐⭐⭐⭐⭐ 最高可行性，最符合业务逻辑**

#### 动机
不同物资的采购是**不独立的**——一个输变电项目通常同时采购电缆、变压器、绝缘子、避雷器等多种物资。当前的"每个物资独立建模"浪费了跨物资的关联信息。

#### 数据层次结构
```
                    国网物资总需求 (Level 0)
                    /          |          \
             变电类           线路类        配电类 (Level 1)
           /   |   \        /   |   \      /   |   \
      变压器  GIS  断路器   电缆 导线 铁塔  避雷器 绝缘子 开关柜 (Level 2)
          (3种目标物资分布在不同Level 2分类下)
```

#### 架构
```
物资C的月度需求量
      │
      ├── 自下而上: 用物资C自身的历史+因子预测 (Base forecast)
      │
      ├── 自上而下: 预测 Level 1 (大类总量) → 按历史比例分配到物资C
      │
      ├── 中间层: 预测 Level 2 (同大类所有物资) 的联合分布
      │
      └── 调和层 (Reconciliation):
            MinT / ERM / BU 最优调和矩阵
            ŷ_reconciled = S × G × ŷ_base
            (S: 层次映射矩阵, G: 调和矩阵)
```

#### 新颖性
- 层次化预测在零售和旅游领域广泛应用，**在电力物资领域为零**
- "大类总量约束 → 物资级分配" 完美匹配国网批次招标的业务逻辑
- 调和矩阵 (MinT / ERM) 从统计学角度最优

#### 发表可行性
- **高**：跨领域方法迁移 + 业务逻辑自洽
- 目标期刊：*IEEE Trans. Power Systems* / *Energy Economics*

### 方案3：PatchTST + 外生变量通道

**⭐⭐⭐⭐ 高可行性**

#### 动机
PatchTST (Nie et al., ICLR 2023) 是目前时间序列预测的 SOTA 之一。它将序列切分为 patch（子序列片段），用 Transformer 学习 patch 间的依赖关系。核心优势：
- 通道独立性 (Channel Independence)：每个变量独立编码
- 自监督预训练：可通过掩码自编码器预训练

但 PatchTST **不支持外生变量**(external regressors)——这正是我们的切入点。

#### 架构
```
标准 PatchTST (未修改)              本文改进: PatchTST + Exog
┌──────────────────┐              ┌──────────────────────────┐
│ y (单通道)        │              │ y (目标通道)              │
│   ↓               │              │   ↓                       │
│ 切分为patch       │              │ 切分为patch               │
│   ↓               │              │   ↓                       │
│ Patch Embedding   │              │ Patch Embedding            │
│   ↓               │              │   ↓                       │
│ Transformer Enc   │              │ ┌──────────────────────┐  │
│   ↓               │              │ │ 外生变量融合层 ← 新增 │  │
│ Flatten + Head    │              │ │ Exog₁..Exog_K 通过   │  │
│   ↓               │              │ │ Cross-Attention注入  │  │
│ 预测              │              │ └──────────────────────┘  │
│                   │              │   ↓                       │
│                   │              │ Transformer Enc            │
│                   │              │   ↓                       │
│                   │              │ Flatten + Head             │
│                   │              │   ↓                       │
│                   │              │ 预测                       │
└──────────────────┘              └──────────────────────────┘

改进点: 在 Patch Embedding 和 Transformer 之间
插入 Cross-Attention 层, 让每个 patch 的表示
能被当月的外部因子 (铜价/铝价/用电量/批次密度) 调制
```

#### 新颖性
- PatchTST 的"外生变量注入"方法**尚未在电力预测领域发表**
- 解决了 PatchTST 原论文不支持外部因子的已知限制
- 通道独立性 = 每个因子独立编码，避免因子间的虚假相关

#### 发表可行性
- **中高**：技术贡献明确（方法论改进），但需要仔细的消融实验

### 方案4：Mamba (State Space Model) 用于电力物资时序

**⭐⭐⭐⭐ 高新颖性**

#### 动机
Mamba (Gu & Dao, 2023) 和 S4 (Gu et al., 2022) 是结构化状态空间模型(SSM)，具有：
- O(N) 的序列长度复杂度 (vs Transformer 的 O(N²))
- 在长序列 (≥100步) 上优于 Transformer
- 对稀疏/不规则采样数据更鲁棒

电力物资序列的特点正是长度适中但**稀疏**，Mamba 的连续时间特性在处理缺失值/零值方面可能优于离散 Transformer。

#### 架构
```
Mamba + VMD 混合预测

    y_train (69个月)
        │
        ├── VMD 分解 → K个IMF
        │       │
        │       └── 每个 IMF → Mamba Block (单独的SSM)
        │                        │
        │                        ├── State Space (A,B,C,D)
        │                        ├── 输入投影 + 卷积
        │                        └── 选择性扫描 (Selective Scan)
        │
        └── 外部因子 → Mamba + Cross-Attention
                │
                └── 预测每个IMF的测试期值 → 求和 → y_pred
```

#### 新颖性
- Mamba/S4 **在电力物资预测领域是空白**
- "SSM + VMD" 替代 "Transformer + VMD" 是自然的架构演进
- 选择性扫描(Selective Scan)机制可以自适应关注重要的历史月份

#### 发表可行性
- **中高**：Mamba 是非常热门的新架构（2023-2024），任何领域的首次应用都有发表价值
- 挑战：需要充分的对比实验（vs LSTM, vs Transformer, vs Informer）

### 方案5：N-HiTS (Neural Hierarchical Interpolation) 处理间歇性

**⭐⭐⭐⭐ 高可行性**

#### 动机
N-HiTS (Challu et al., AAAI 2023) 是目前长时序预测的最新 SOTA 之一。它通过**层次化插值**在不同时间尺度上建模：
- Level 1：学习粗粒度的趋势（年周期）
- Level 2：学习中粒度的季节模式（季度周期）
- Level 3：学习细粒度的波动（月度）

这恰好对应电力物资的**多时间尺度特征**：
- 年尺度：投资计划的年度节奏
- 季度尺度：批次招标的季度分布
- 月度尺度：具体的采购执行

#### 架构
```
输入: y (69个月历史需求量)

┌──────────────────────────────────────────┐
│ Stack 1: 粗粒度 (pooling ratio=12)        │
│   MaxPool(x, kernel=12) → MLP → 上采样    │
│   学习: 年度趋势                           │
├──────────────────────────────────────────┤
│ Stack 2: 中粒度 (pooling ratio=4)         │
│   MaxPool(x, kernel=4) → MLP → 上采样     │
│   学习: 季度模式 (批次节奏)                  │
├──────────────────────────────────────────┤
│ Stack 3: 细粒度 (pooling ratio=1)         │
│   MLP → 学习月度精细波动                    │
└──────────────────────────────────────────┘
                    │
                    ▼
    Σ(stack_1 + stack_2 + stack_3) → 预测
                    │
      ┌─────────────┴─────────────┐
      ▼                           ▼
  点预测                      分位数预测
 (Point Forecast)          (P10, P50, P90)
```

#### 与外部因子结合
N-HiTS 原生不支持外部因子。引入方式：
- **方案A**：将外部因子作为额外输入通道，通过卷积+池化与目标通道融合
- **方案B**：外部因子单独通过 MLP 编码，注入到每个 Stack 的输出

#### 新颖性
- N-HiTS **从未用于电力采购预测**
- 层次化结构天然匹配"年度投资→季度批次→月度执行"的业务逻辑
- 比 VMD-LSTM 简单得多（无需信号分解+参数优化），但表达能力可能更强

#### 发表可行性
- **中高**：方法简洁 + 领域首创 + 可解释的层次化预测

### 方案6：多任务学习 (Multi-Task Learning) 跨物资共享表示

**⭐⭐⭐ 中等可行性**

#### 动机
三种物资（避雷器、互感器、绝缘子）的需求受**相同的宏观因素影响**（投资计划、经济周期、季节），但各自有不同的微观特征。

#### 架构
```
          共享特征提取层 (Shared Backbone)
        ┌─────────────────────────────────┐
        │  LSTM / TCN / Transformer        │
        │  (从历史需求 + 外部因子提取共享表示)  │
        └───────────┬─────────────────────┘
                    │
        ┌───────────┼───────────┐
        ▼           ▼           ▼
   Task Head 1  Task Head 2  Task Head 3
   (避雷器)      (互感器)      (绝缘子)
        │           │           │
        ▼           ▼           ▼
    ŷ_arrester  ŷ_cvt       ŷ_insulator

   Loss = α·MSE(avoidance) + β·MSE(cvt) + γ·MSE(insulator)
          + λ·Orthogonality(Head1, Head2, Head3)
```

#### 新颖性
- 多任务学习在 CV/NLP 成熟，**在电力物资预测中未被使用**
- 正交约束保证各 Task Head 学到物资特有的模式

#### 可行性/发表性
- **中等**：需要3种以上物资才有意义（目前恰好3种），技术贡献面偏窄

### 方案7：批次事件驱动的 Conditional Forecasting

**⭐⭐⭐⭐⭐ 实用价值最高**

#### 动机
**这是最能解决实际问题的方案**。ECP 批次安排是公开信息（预安排在每年初发布），我们可以提前知道"3月会有输变电第二次设备招标"、"6月有智能电表第一次招标"。

当前所有模型都把批次信息作为隐式的季节模式来学习。但如果把批次预安排作为一种**确定性的未来事件**输入模型，预测精度应该大幅提高。

#### 架构
```
Conditional CatBoost / TFT

训练时的特征:
┌───────────────────────────────────────────────┐
│ 对于时间步 t (例如 2025-03):                    │
│                                                │
│ - 历史需求 lag1,lag2,lag3,lag6,lag12            │
│ - 外部因子 copper_price, industrial_elec...     │
│ - ★ 批次事件特征 (NEW):                          │
│   · is_bid_batch_month: 本月是否有招标批次 (0/1)  │
│   · batch_type: one-hot(输变电/特高压/电源/数字化) │
│   · days_since_last_batch: 距上次批次天数         │
│   · days_until_next_batch: 距下次批次天数         │
│   · cumulative_batches_this_year: 本年累计批次数  │
│   · batch_phase: 本年度第几次批次 (1st/2nd/...)    │
└───────────────────────────────────────────────┘

测试时的预测:
┌───────────────────────────────────────────────┐
│ 预测 2026-09 的需求:                            │
│                                                │
│ 我们可以提前知道:                               │
│ - 2026-09 有输变电第5次设备招标 ✓                │
│ - 距离上次批次 45 天 ✓                          │
│ - 本年第 14 个批次 ✓                            │
│                                                │
│ → 模型可以利用这些确定性信息                     │
│   来预判 9 月会有需求波峰                        │
└───────────────────────────────────────────────┘
```

#### 新颖性
- "确定性的未来事件作为预测条件" 这一思路**在电力物资预测中从未被提出**
- 比当前 research 中流行的 VMD-LSTM 等黑箱方法**更实用，更具解释性**
- 可以单独发表也可以嵌入任何模型框架

#### 发表可行性
- **极高**：创新点清晰,实用性强,数据和代码完全可控
- 投稿方向：*Electric Power Systems Research* / *IEEE Trans. Smart Grid* / *Applied Energy*

## 四、推荐实施路线

### 近期 (立即可做): 方案7 批次事件驱动
- 数据已就绪（ECP DB 中有批次信息）
- 无需改模型架构，只需加特征
- CatBoost + 批次特征，预计 R² 从 0.15~0.49 → **0.5~0.7**

### 中期 (1-2周): 方案2 层次化预测
- 需要构建物资分类层次树
- MinT 调和在 `scikit-hts` 或 `hierarchicalforecast` 库中直接可用
- 论文创新点：跨领域方法迁移

### 长期 (1月+): 方案1 或 方案4
- Croston+TFT 或 Mamba-VMD
- 发表价值高，但实现复杂度也高
- 适合作为期刊论文的核心方法
