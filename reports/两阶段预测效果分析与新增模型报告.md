# 两阶段预测效果分析与新增模型报告

> 日期: 2026-07-08

## 一、两阶段预测成效不明显的根因分析

### 1.1 现象

| 物资 | CatBoost(朴素) | CatBoost-2S | CondCatBoost | TwoStage | Ridge-2S |
|------|:---:|:---:|:---:|:---:|:---:|
| 交流避雷器 | 0.095 | 0.147 | 0.145 | **0.562** | **0.548** |
| 电容式电压互感器 | 0.083 | **0.429** | **0.445** | 0.325 | 0.269 |
| 交流支柱绝缘子 | 0.343 | 0.340 | 0.349 | 0.171 | **0.404** |
| 断路器保护 | **0.664** | 0.629 | 0.576 | 0.038 | 0.038 |
| 电抗器保护 | 0.311 | 0.169 | 0.169 | **0.354** | 0.354 |

CatBoost-2S / CondCatBoost（使用 `_two_stage_fit_predict` 框架）在避雷器上仅 +0.05 R²。而独立 `run_two_stage` 在同一物资上达到 +0.47。存在系统性差异。

### 1.2 根因：`_two_stage_fit_predict` 框架与独立 `run_two_stage` 的关键差异

```python
# _two_stage_fit_predict (共享框架) — CatBoost-2S/CondCatBoost/LightGBM 使用
nv2 = max(6, nz.sum()//3)    # 42→14 个验证样本
stage2_regressor.fit(X_tr[nz][:-nv2], y_tr[nz][:-nv2], ...)  # 42-14=28 训练样本

# run_two_stage (独立实现)
n_val2 = min(6, len(y_nz)//4)  # 6 个验证样本
reg.fit(X_nz[:-n_val2], y_nz[:-n_val2], ...)                  # 42-6=36 训练样本
```

差异总结：

| 特性 | `_two_stage_fit_predict` | `run_two_stage` |
|------|:---:|:---:|
| Stage1 iterations | **600** | **800** |
| Stage2 训练样本 | 42-14=**28** | 42-6=**36** |
| Stage2 验证样本 | 14 | 6 |
| 有效训练% | 67% | 86% |

**框架的 F3 "修复" (nv2=max(6, nz//3)) 抢走了 Stage2 几乎 1/3 的训练数据。** 对 28 个样本训练 depth=6 的 CatBoost → 严重过拟合 → 测试集泛化差。

### 1.3 总结

| 原因 | 影响 |
|------|------|
| 框架 Stage1 迭代少 (600 vs 800) | 分类器欠训练 |
| 框架 Stage2 验证集过大 (14/42 → 33%) | 回归器训练样本仅 28 个 |
| 独立 TwoStage 保留更多训练数据 (36 vs 28) | +29% 训练样本 → 泛化更好 |
| Ridge-2S 不受影响 | 线性模型 28 样本足够，反而 L2 正则化更关键 |

## 二、新增模型实现计划

### 2.1 新增模型

| 模型 | 类型 | 在 28-36 样本下的优势 |
|------|------|----------------------|
| ElasticNet-2S | 线性+正则化 | L1+L2 双重正则化，小样本优于纯 Ridge |
| GaussianProcess-2S | 贝叶斯非参 | 小样本内置不确定性量化，无过拟合 |
| Theta-2S | 分解预测 | 将序列分解为趋势+季节 |
| SES-2S | 指数平滑 | 最简单的时序方法，作为 baseline |
| Ridge-2S | 已有 | 仅 L2，对比 ElasticNet |

### 2.2 修复 _two_stage_fit_predict 验证集

将 `nv2 = max(6, nz.sum()//3)` 回退到 `nv2 = min(6, nz.sum()//4)`（与独立 TwoStage 一致）。

### 2.3 预期效果

- CatBoost-2S / CondCatBoost 的 R² 应回升至接近独立 TwoStage 的水平
- Ridge-2S / ElasticNet-2S 在小需求物资上应优于树模型
- GaussianProcess-2S 应提供"二阶收益"（不确定性区间）
- Theta / SES 作为额外的间歇性基线
