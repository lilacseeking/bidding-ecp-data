## 二、低确定性步骤的替代方案

### 步骤8: 特征工程重构 (确定性 35%)

**问题**: 当前 `preprocess_data()` 有三个致命数据泄露（MinMaxScaler全量fit、lag全量roll后切分、rolling全量计算后切分），需要完全重写。

**替代方案A (推荐)**:
- 重构为时序安全管线，使用 `sklearn.pipeline.Pipeline` + `sklearn.model_selection.TimeSeriesSplit`
- Scaler仅对训练集 `.fit_transform()`，测试集 `.transform()`
- lag特征在切分后的训练集中计算：`df_train['lag1'] = df_train['target'].shift(1)`，测试集的lag值来自训练集历史窗口
- rolling统计同样仅在训练集计算：`df_train['roll3_mean'] = df_train['target'].rolling(3).mean()`

**替代方案B**:
- 使用 `tsfresh` 或 `feature-engine` 库的时序安全特征生成函数
- `feature-engine.timeseries.forecasting.LagFeatures` 原生支持训练集/测试集分别处理
- 减少手写特征工程代码量，降低出错概率

**降风险措施**: 重构后对synthetic数据做验证（已知真实值的模拟序列），确保无泄露后R² = 真实预测能力。

---

### 步骤10: VMD方法学重新设计 (确定性 25%)

**问题**: 当前4个VMD模型全部存在循环论证（VMD分解 y_train → IMF作为特征 → 预测 y_train，由于 ΣIMF≈y，模型只需学习恒等映射）。CatBoost因正则化得到 R²=0.07，SVR无正则化得到 R²=0.99（虚假精度）。所有VMD模型必须废弃方法学但保留基础函数。

**替代方案A (推荐): VMD去噪+外部因子预测**
```
VMD(y_train) → {IMF_残差, IMF_1, IMF_2, ..., IMF_K}
筛选高频噪声分量（基于中心频率阈值或Pearson相关系数）
y_denoised = Σ(保留的IMFs)  # 重构去噪后的信号
X_train_aug = [y_denoised_lag1, y_denoised_lag3, ..., external_factors]
CatBoost(X_train_aug) → y_test
```
- 优点: 不形成循环论证。VMD的作用是"去噪"而非"生成特征"。模型预测的还是原始y，但不是用IMF作为特征。
- 弱点: 如果VMD去噪后信号与原始y的差异不大，提升有限。但这是诚实的方法。
- 论文卖点: "VMD用于电力物资需求序列的去噪预处理"而非"VMD用于生成预测特征"

**替代方案B: VMD分频预测（独立预测各IMF，最终加总）**
```
VMD(y_train) → {IMF_残差, IMF_1, ..., IMF_K}
对每个IMF_k:
    训练 ARIMA/Transformer/CatBoost(IMF_k的历史值, external_factors) → 预测 IMF_k_test
y_pred = Σ(各IMF的预测值)
```
- 优点: 不形成循环论证。每个IMF的预测仅使用该IMF的历史和外部因子，不使用其他IMF。
- 弱点: IMF分量之间不完全独立，独立预测忽略跨IMF的依赖关系。单IMF预测可能误差大。
- 论文中有发表先例: 多个类似的分频预测框架已被验证（如VMD+LSTM各分量独立预测再求和）。

**替代方案C: 不使用VMD，直接CatBoost+增强因子**
- 如果方案A和B的效果均不理想，论文可在增强因子 + CatBoost/XGBoost/LightGBM + TFT/N-HiTS 等架构上做对比，完全放弃VMD。
- 论文卖点: "外部市场因子首次引入电力物资需求预测" + "全国聚合ECP数据的大规模实证" 本身已有发表价值。

### 步骤14: 评估体系设计 (确定性 45%)

**问题**: 当前仅单次69/12分割 + 4个基础指标（MSE/RMSE/MAE/R²），测试集仅12个月，一个异常月可显著改变R²。缺少时序交叉验证、统计显著性检验、零值处理指标、标准基线。

**替代方案A (推荐)**:
```
评估体系 = {
    交叉验证: TimeSeriesSplit(n_splits=5, test_size=12) expanding window,
    指标: [RMSE, MAE, R², MASE, sMAPE],
    统计检验: Diebold-Mariano Test (模型间差异显著性),
    基线: [Naive Seasonal, SARIMA, CatBoost(factors only)],
    稳定性: 不同测试窗口长度(6/12/18个月)的R²方差
}
```
- MASE: 处理零值月的标准指标，以Naive Seasonal为基准缩放误差
- sMAPE: 对称MAPE，有界（0-200%），不受零值影响
- Diebold-Mariano: 回答"模型A是否统计显著优于模型B"，审稿人常见要求

**替代方案B**: 回测框架
```
每个月作为一次预测原点 → 用此前所有数据训练 → 预测未来1-12个月 → 计算多步预测误差
```
- 更接近真实预测场景
- 能评估"预测提前期"对精度的影响（1步/3步/6步/12步预测）
- 实现复杂度较高但论文说服力强

### 可以发表的核心理由

1. **差异化定位有文献支撑**：4个方向均无直接冲突（外部市场因子、VMD+Transformer+CatBoost组合、农历编码、两阶段框架），有明确的空白可以填补。

2. **数据工程基础扎实**：数据采集管线（API采集、Excel解析、层叠解析框架）架构正确，3个缺陷修复方案明确（2-3天工作量），修复后的数据质量可支持论文发表。

3. **代码资产可保留率高**：18个代码模块中，9个可直接保留，4个需修改后保留，仅4个VMD模型和1个特征工程模块需废弃/重写。Transformer和VMD基础组件质量好，可复用。

4. **影响因子增强有明确路线图**：P0因子（批次事件+YoY衍生+春节距天数）3人日可完成，预期CatBoost baseline R²可从0.15-0.49提升至0.50-0.65，达到论文可发表水平。

5. **"否定性结果"本身有发表价值**：即使外部市场因子预测力不足，"首次系统评估外部市场因子对电力物资需求预测的贡献"也是一项有价值的发现。
